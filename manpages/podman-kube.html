    NAME

       podman-kube - Play containers, pods or volumes based on a structured input file

    SYNOPSIS

       podman kube subcommand

    DESCRIPTION

       The kube command will recreate containers, pods or volumes based on the input from a structured (like YAML) file input.  Containers will be automatically started.

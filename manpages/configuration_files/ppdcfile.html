    ppdcfile

    ppdcfile(5) OpenPrinting ppdcfile(5)

    NOM

       ppdcfile - Format de fichier source ppd pour compilateur CUPS (obsolète)

    DESCRIPTION

       Le  compilateur  PPD de CUPS lit les fichiers de métadonnées qui contiennent un ou plusieurs fichiers PPD devant être créés par ppdc(1). Cette page de manuel
   décrit de manière succincte les mots-clefs pris en charge et devrait être utilisée de concert avec l’aide en ligne de CUPS.

   Le fichier source est au format texte pur ASCII et peut être édité avec n’importe quel éditeur de texte. Les commentaires sont pris en  charge  en  utilisant
   les mécanismes des langages C (/* ... */) et C++.

   Les  informations  sur  les  pilotes  d’imprimante  peuvent  être  groupées  et partagées en utilisant des accolades ({ ... }) ; les fichiers PPD sont écrits
   lorsqu’une directive PCFileName a été définie.

   Les directives peuvent être placées n’importe où sur une ligne et sont suivies par une ou plusieurs valeurs. Ce qui suit constitue la  liste  des  directives
   disponibles et les valeurs acceptées :

   #define nom valeur

   #elif {nom | valeur}

   #else

   #endif

   #font nom encodage "version" jeu_de_caractères état

   #if {nom | valeur}

   #include <nom_fichier>

   #include "nom_fichier"

   #media nom largeur hauteur

   #media "nom/texte" largeur hauteur

   #po locale "nom_fichier"

   Attribute nom "" valeur

   Attribute nom mot-clef valeur

   Attribute nom "mot-clef/texte" valeur

   Choice nom "code"

   Choice "nom/texte" "code"

   ColorDevice valeur_booléenne

   ColorModel nom espace_colorimétrique ordre_des_couleurs compression

   ColorModel "nom/texte" espace_colorimétrique ordre_des_couleurs compression

   ColorProfile résolution/type_média gamma densité_matrice

   Copyright "texte"

   CustomMedia nom largeur hauteur gauche bas droite haut "code_dimension" "code_région"

   CustomMedia "nom/texte" largeur hauteur gauche bas droite haut "code_dimension" "code_région"

   Cutter valeur_booléenne

   Darkness température nom

   Darkness température "nom/texte"

   DriverType type

   Duplex type

   Filter type_mime coût programme

   Finishing nom

   Finishing "nom/texte"

   Font *

   Font nom encodage "version" jeu_de_caractères état

   Group nom

   Group "nom/texte"

   HWMargins gauche bas droite haut

   InputSlot position nom

   InputSlot position "nom/texte"

   Installable nom

   Installable "nom/texte"

   LocAttribute nom "mot-clef/texte" valeur

   ManualCopies valeur_booléenne

   Manufacturer "nom"

   MaxSize largeur hauteur

   MediaSize nom

   MediaType type nom

   MediaType type "nom/texte"

   MinSize largeur hauteur

   ModelName "nom"

   ModelNumber numéro

   Option nom type section ordre

   Option "nom/texte" type section ordre

   PCFileName "nom_fichier.ppd"

   Resolution espace_colorimétrique octets_par_couleur row-count row-feed row-step nom

   Resolution espace_colorimétrique octets_par_couleur row-count row-feed row-step "nom/texte"

   SimpleColorProfile résolution/type_média densité densité_jaune densité_rouge gamma ajustement_rouge ajustement_vert ajustement_bleu

   Throughput pages_par_minute

   UIConstraints "*Option1 *Option2"

   UIConstraints "*Option1 Choix1 *Option2"

   UIConstraints "*Option1 *Option2 Choix2"

   UIConstraints "*Option1 Choix1 *Option2 Choix2"

   VariablePaperSize valeur_booléenne

   Version numéro

    NOTES

       Les  fichiers  PPD sont obsolètes et ne seront plus gérés dans une future version de CUPS. Les imprimantes ne gérant pas IPP peuvent être gérées avec des ap‐
   plications telles que ippeveprinter(1).

    NOM

       printers.conf - Fichier de configuration des imprimantes CUPS

    DESCRIPTION

       Le fichier printers.conf définit les imprimantes locales disponibles. Il est habituellement situé dans le répertoire /etc/cups et est tenu à jour par le pro‐
   gramme cupsd(8). Ce fichier n’est pas supposé être édité ou géré par l’utilisateur.

    NOTES

       Le nom, l’emplacement et le format de ce fichier sont des détails d’implémentation qui changeront dans les prochaines versions de CUPS.

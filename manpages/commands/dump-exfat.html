    NAME

       dump.exfat - Show on-disk information of exfat filesystem

    SYNOPSIS

       dump.exfat device
   dump.exfat -V

    DESCRIPTION

       dump.exfat Print on-disk information from given device that formatted by exFAT filesystem.

    OPTIONS

       -V     Prints the version number and exits.

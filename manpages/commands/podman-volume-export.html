    NAME

       podman-volume-export - Exports volume to external tar

    SYNOPSIS

       podman volume export [options] volume

    DESCRIPTION

       podman  volume  export  exports  the contents of a podman volume and saves it as a tarball on the local machine. podman volume export writes to STDOUT by default and can be redi‐
   rected to a file using the --output flag.

   Note: Following command is not supported by podman-remote.

   podman volume export [OPTIONS] VOLUME

    OPTIONS --help

       Print usage statement

    --output, -o=file

       Write to a file, default is STDOUT

    EXAMPLES

              $ podman volume export myvol --output myvol.tar

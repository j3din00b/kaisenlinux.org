    NAME

       podman-pod-exists - Check if a pod exists in local storage

    SYNOPSIS

       podman pod exists pod

    DESCRIPTION

       podman  pod  exists checks if a pod exists in local storage. The ID or Name of the pod may be used as input.  Podman will return an exit code of 0 when the pod is found.  A 1 will
   be returned otherwise. An exit code of 125 indicates there was an issue accessing the local storage.

    EXAMPLES

       Check if a pod called web exists in local storage (the pod does actually exist).

          $ sudo podman pod exists web
          $ echo $?
          0
          $

   Check if a pod called backend exists in local storage (the pod does not actually exist).

          $ sudo podman pod exists backend
          $ echo $?
          1
          $
